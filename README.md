# HPL101 — Study Notes

Self-contained HTML study notes for **HPL101 (How People Learn)**, Harvard GSE.
Open `index.html` for the dashboard; each unit links to its own notes page.

## Live site

Once GitHub Pages is enabled (Settings → Pages → Deploy from branch `main` / root),
this is served at:

```
https://<your-username>.github.io/<repo-name>/
```

## Contents

- `index.html` — dashboard / hub (auto-detects which unit notes exist)
- `cognitivism-notes.html` — Unit 2.1.2 · Cognitivism
- _more units added over time_

## Notes

These are personal study notes that paraphrase course material for my own review.
The `.nojekyll` file tells GitHub Pages to serve the files as-is.
