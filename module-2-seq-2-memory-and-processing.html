Memory & Active Processing

Working memory is a bottleneck. Learning means managing what passes through it — and what we actively do with it.

working & long-term memory cognitive load memory strategies active processing

01 The core idea

Every performance — a sport, a test, a job interview, making breakfast — taxes our memory. But we can only handle so much information at once, especially under stress. As educators we ask: in this situation, are we being mindful of learners' memory capacities, and what can we do to optimize memory?

Think of human memory like a computer's bandwidth and storage. Bandwidth = how much we can process at one time (this is working memory, and it is very limited). Storage = how much we can keep over time (this is long-term memory, and it is vast). Good teaching protects the narrow bandwidth so more can reach the deep storage.

02 Types of memory & knowledge · 2.2.1

Human memory has three key components, and the knowledge stored in long-term memory comes in two flavors.

The information-processing model. Only what you attend to enters working memory; only what you process reaches long-term storage.

Two kinds of knowledge in long-term memory

Declarative knowing what

Facts, information, and events — expressed through recognition and recall. The conscious "what."

e.g. Recalling traffic laws for a written driver's-license exam; naming math principles.

Procedural knowing how

How to perform a task — actions, skills, processes. With practice it becomes fluent and runs without taxing working memory.

e.g. An expert chef cooking quickly while chatting; a driver shifting feet between pedals automatically.

↳ Most real tasks mix both. Experts proceduralize routines to free up working memory (Ramsay vs. amateur cook). Catch faulty procedures early — bad habits are hard to relearn later. Education also teaches learners to integrate declarative and procedural knowledge.

Worked examples. To build procedural knowledge, studying a few worked examples — and explaining why each step is right or wrong — beats grinding through many problems. It forces "minds-on" thinking, reduces extraneous detail, and (per HGSE's Jon Star & Rittle-Johnson) comparing two solved approaches deepens both procedural skill and conceptual understanding.

03 Cognitive load · 2.2.2

Cognitive load is the total amount of information processed in working memory at any one time — and it is limited. (It's actually fortunate it's limited: people with "highly superior autobiographical memory" / HSAM can't forget the trivial, which interferes with focusing on the present.) Total load is theorized to have three types, and they trade off — a lot of one leaves less room for the other two.

IntrinsicProcessing the task inherently requires. Fixed by the task's difficulty — can't be changed. Spelling "chaperone" > spelling "mom."
ExtraneousProcessing spent on unnecessary components. Should be reduced by good design — e.g. wordy math problems vs. clean equations.
GermaneProcessing that builds learning and understanding; key for storing in long-term memory. Should be maximized.

The teacher's move: intrinsic load is set by the task. So if a task is demanding, cut extraneous load (remove clutter and distractions) to free mental resources for germane load — understanding the problem and how it connects to others. Provide scaffolds that break tasks into well-sequenced chunks and link new info to prior learning.

Choking under pressure. Beilock's soccer-dribbling study: making expert players attend to step-by-step details of an already-proceduralized skill made them slower and more error-prone. Anxiety and worry consume working memory, so even champions can "choke" — performance suffers when working memory is occupied by something other than the task.

04 Strategies for improving memory · 2.2.3

Working memory holds only ~7–10 items at once. These five strategies stretch and strengthen what we can process and retain.

Group connected pieces of information so they're stored or processed as single units.

Chunking Turns 12 letters (EPAHPLGSEHDE) into 4 meaningful chunks (EPA·HPL·GSE·HDE). Often implicit, it leans on long-term memory (e.g. sorting names into athletes / presidents / singers) to offload the limited working memory.

Spread study out over time instead of cramming it into one block.

Spacing The spacing effect: one hour a day for a week (Zara) beats seven hours one Saturday (Jacques). Distributed practice forces repeated review, new associations, and elaboration each time.

Alternate between topics rather than studying one in a long block.

Interleaving Mixing problems (ABAB… or random) forces learners to spot each problem's deeper structure and choose a strategy, boosting cognitive flexibility. Baseball hitters who practiced randomly-ordered pitches outperformed blocked-practice hitters in gamelike conditions. Interleaving also builds in spacing and retrieval.

Practice retrieving information instead of just rereading it.

Testing effect Roediger & Karpicke: rereading wins after 5 minutes, but recalling wins after 2 days and a week. Each act of retrieval makes the next easier and strengthens cue→answer links. Pair with feedback — otherwise learners cement wrong answers. Strongest for simpler declarative facts.

Teach or coach someone else what you're trying to learn.

Protégé effect Teaching drives active retrieval (the testing effect again), forces you to reorganize knowledge, and engages metacognition. Merely expecting to teach can boost learning.

05 Active processing · 2.2.4

The Ball-Passing (selective-attention) task shows that only what you attend to enters working memory from the sensory register — and attention is an active process, never passive. But attention is only the first step.

Active processing = manipulating knowledge in working memory: linking it to what you already know, revising it, relating it to other knowledge, or applying it. Simply adding information to working memory won't change how learners reason — processing turns information into usable knowledge. This, too, cannot happen passively.

Design heightens the odds. Learners decide whether to process actively, but learning environments can be designed to make it more likely. Re-representing a problem in a new format, explaining it to someone else, or relating it to earlier learning all force active re-working. This is what educators mean by "active learning."

Notes & flashcards: handwriting notes (selecting, paraphrasing) beats verbatim typing for deep processing; routine highlighting doesn't reliably help. Flashcards work if they promote processing — making your own, elaborating answers, delaying feedback — rather than rote recall.

06 Glossary — quiz yourself first

Try to define each term before you reveal it.

working memory
Part of the memory system that facilitates planning, comprehension, reasoning, and problem-solving by allowing the short-term retention and manipulation of a limited amount of information at any one time. The "bandwidth" of the mind — narrow, ~7±2 chunks, where meaning is made.
declarative knowledge
Learning of facts, information, and events, often expressed in terms of recognition and recall — "knowing what." Reciting traffic laws for a written test.
procedural knowledge
Learning related to how to perform a task — "knowing how." With practice it proceduralizes and runs without taxing working memory. An expert chef cooking fluently while talking.
worked example
A step-by-step demonstration of how to complete a task or solve a problem; studying and explaining these builds procedural knowledge more efficiently than repetitive practice. Comparing two solved algebra problems and explaining why each step works.
cognitive load
The total amount of information that can be processed in our working memory at any given time. It is limited; its three types trade off against one another.
intrinsic cognitive load
The information-processing capacity necessary for performing a task. It is inherent to the nature of the task and cannot be changed. Spelling "chaperone" carries more intrinsic load than "mom."
extraneous cognitive load
The information-processing capacity dedicated to unnecessary components of a task. It can and should be reduced through thoughtful design. A wordy math word-problem adds extraneous load over a clean equation.
germane cognitive load
The information-processing capacity essential for learning and understanding to result from performing a task. Important for storing in long-term memory; should be maximized.
chunking
Grouping together connected pieces of information during learning so they can be stored or processed as single concepts, leaning on long-term memory to relieve working memory. EPA · HPL · GSE · HDE — four chunks instead of twelve letters.
spacing
Segmenting information to be learned, reducing the demands on working memory at any one time and promoting retention. The spacing effect: distributed study beats massed study.
interleaving
A process of mixing the order of subjects or topics to improve learning. Forces learners to discriminate deep structure and select strategies, and promotes cognitive flexibility.
testing effect
The facilitation of learning by performing repeated tests or acts of retrieval of what has been learned. Each act of retrieval makes future retrieval easier — best paired with feedback.
protégé effect
The idea that explaining or teaching a skill, task, or subject to another can deepen the teacher's own retention and conceptual understanding.
active processing
The act of manipulating knowledge in working memory: modifying it, revising it, linking it to previous knowledge, or applying it. It cannot happen passively.

★ Remember this

  • Working memory is the bottleneck. It's tiny (~7±2 chunks); long-term memory is vast. Teach to protect the bottleneck.
  • Attention gates learning. Only what you actively attend to enters working memory — and attention is never passive.
  • Manage the three loads. Intrinsic is fixed; cut extraneous, maximize germane. A lot of one leaves less for the others.
  • Declarative vs. procedural. "Knowing what" vs. "knowing how"; proceduralizing frees working memory, but pressure can make experts choke.
  • Strong study strategies: chunking, spacing, interleaving, retrieval practice (with feedback), and teaching others.
  • Active processing turns info into knowledge. Link, revise, relate, apply — that's what "active learning" designs for.
